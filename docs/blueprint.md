# **App Name**: NeuroRadX

## Core Features:

- User Roles & Access: User authentication and role management based on subscription level and specialization (student, resident, fellow, specialist).
- Question Bank Management: Structured question bank with various question types (MCQ, case-based, image-based), searchable by topic, subtopic, and difficulty level.
- Image Handling: Display high-resolution medical images with zoom and pan capabilities, potentially integrating a basic DICOM viewer or conversion for previews.
- Progress Tracking: Track user progress (scores, time spent) and provide personalized feedback based on performance in different categories.
- Study Modes: Implement different study modes (Exam, Tutor, Flashcard) with customizable preferences and scoring.
- Personalized Learning Tools: Enable users to bookmark questions, add personal notes, and search the question bank effectively.
- Offline Access: Implement offline access to pre-downloaded question blocks and images for studying on the go.

## Style Guidelines:

- A muted teal (#45A0A2) evoking a sense of calm focus, conducive to study and concentration, yet not stereotypical for medical apps.
- Light grey (#F0F2F5) to ensure readability and reduce eye strain during long study sessions.
- A soft, desaturated blue (#77B5FE) to highlight key UI elements and interactive components, creating visual interest without being distracting.
- Clean, sans-serif fonts optimized for readability on digital screens.
- Simple, professional icons to represent topics, question types, and study modes.
- Clean and intuitive layout that allows the user to focus on the content.
- Subtle animations and transitions to provide feedback on user interactions.
- Dark mode color scheme for low-light environments.
